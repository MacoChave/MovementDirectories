Instrucciones de uso
1.- El archivo "tarea1.c" debe estar en la misma carpeta donde se encuentre la biblioteca de musica, no dentro de esta.

2.- Los comandos a los que responde el programa son 
	
	s     		-> organizar la biblioteca
	n     		-> no organizar la biblioteca
	mostrar		-> muestra los archivos de la carpeta actual
	mover a (nombre)-> moverse a la carpeta (nombre)
	abrir (nombre*)	-> imprimir tags de la cancion (nombre)
	azar		-> elige un artista al azar y muestra sus canciones
	volver		-> vuelve a la carpeta anterior
	salir		-> termina el programa

*sin la extension .mp3
Nota: el programa no explota con otras instrucciones (99,9999999% seguro)
